# Create VM Instance use cloud shell and run VM

# create vm us shell

open Cloud shell
--
first you must open cloud shell

copy this command to create instance
--
`gcloud compute instances create <<VM_name>>`

chek your VM
--
`ping <<external_IP>>`

delete your vm
--
`gcloud compute instances delete <<VM_name>>`
